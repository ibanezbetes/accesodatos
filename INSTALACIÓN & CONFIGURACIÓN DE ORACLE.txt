Configuración de la sesión
------------------------------------------
PARA PODER CREAR USUARIOS
SET SERVEROUTPUT ON
ALTER SESSION SET "_ORACLE_SCRIPT" = TRUE;
CREATE USER LOLO IDENTIFIED BY LOLOLAND;
GRANT CONNECT TO LOLO;
GRANT RESOURCE TO LOLO;
GRANT DBA TO LOLO;
CONNECT LOLO/LOLOLAND;

NOTAS:  Hay que tener instalado el Docker Desktop
